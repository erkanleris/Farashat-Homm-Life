# فراشات هوم - Farashat Homm Life

موقع إلكتروني فخم لعرض وبيع منتجات **Homm Life** التركية، يستهدف الجمهور العربي (بلهجة سورية) والجمهور التركي.

## 🌟 الميزات الرئيسية

### 🎨 التصميم والهوية البصرية
- **لوحة ألوان فاخرة:** زهري، ذهبي، أسود
- **وضع ليلي/نهاري:** تبديل سلس بين الوضعين مع تناسق الألوان
- **تصميم متجاوب:** Mobile-First، متوافق مع جميع الأجهزة
- **خطوط احترافية:** دعم العربية والتركية

### 🌐 دعم اللغات
- **العربية:** باللهجة السورية (RTL)
- **التركية:** (LTR)
- **تبديل سلس:** مع حفظ التفضيل

### 🛍️ الكتالوج والمنتجات
- عرض احترافي للمنتجات مع صور عالية الجودة
- نظام فلترة حسب الفئات
- عرض الأسعار والخصومات (حتى 50%)
- ملصق Sale متحرك على المنتجات المخفضة

### 💬 التواصل
- **زر واتساب عائم:** للتواصل السريع
- **صفحة اتصال:** مع نموذج استفسار متكامل
- **إشعارات:** تنبيهات فورية للمالك

### 🔐 لوحة التحكم
- إدارة المنتجات (إضافة، تعديل، حذف)
- إدارة الفئات
- عرض الطلبات والاستفسارات
- حماية الوصول (Admin فقط)

### 📱 تهيئة SEO
- Meta Tags و Open Graph
- robots.txt
- Structured Data جاهز للإضافة
- Sitemap

## 🚀 البدء السريع

### المتطلبات
- Node.js 22+
- pnpm 10+

### التثبيت

```bash
# استنساخ المستودع
git clone https://github.com/erkanleris/Farashat-Homm-Life.git
cd Farashat-Homm-Life

# تثبيت المكتبات
pnpm install

# تشغيل الخادم
pnpm dev
```

الموقع سيكون متاحاً على: `http://localhost:3000`

## 📁 هيكل المشروع

```
farashat-homm-life/
├── client/                 # الواجهة الأمامية (React + TypeScript)
│   ├── src/
│   │   ├── pages/         # الصفحات الرئيسية
│   │   ├── components/    # المكونات المعاد استخدامها
│   │   ├── contexts/      # React Contexts (اللغة، الموضوع)
│   │   ├── lib/           # المساعدات والدوال
│   │   └── index.css      # الأنماط العامة
│   └── public/            # الملفات الثابتة
├── server/                # الخادم الخلفي (Express + tRPC)
│   ├── routers.ts         # tRPC procedures
│   ├── db.ts              # دوال قاعدة البيانات
│   └── _core/             # الإعدادات الأساسية
├── drizzle/               # قاعدة البيانات (Drizzle ORM)
│   └── schema.ts          # تعريف الجداول
└── shared/                # الملفات المشتركة
```

## 🛠️ التكنولوجيا المستخدمة

### الواجهة الأمامية
- **React 19** - مكتبة واجهات المستخدم
- **TypeScript** - لغة البرمجة
- **Tailwind CSS 4** - أنماط CSS
- **Vite** - بناء وتطوير سريع
- **wouter** - التوجيه
- **tRPC** - استدعاء الإجراءات الخلفية

### الخادم الخلفي
- **Express 4** - إطار عمل الويب
- **tRPC 11** - RPC آمن النوع
- **Drizzle ORM** - ORM لقاعدة البيانات
- **MySQL/TiDB** - قاعدة البيانات

### أدوات التطوير
- **Vitest** - اختبارات الوحدة
- **Prettier** - تنسيق الكود
- **TypeScript** - فحص النوع

## 📖 الصفحات الرئيسية

| الصفحة | الرابط | الوصف |
|--------|--------|-------|
| الرئيسية | `/` | صفحة Hero مع معلومات عن المتجر |
| الكتالوج | `/catalog` | عرض جميع المنتجات مع الفلترة |
| الاتصال | `/contact` | نموذج الاستفسار والتواصل |
| لوحة التحكم | `/admin` | إدارة المنتجات (Admin فقط) |

## 🔧 الأوامر المتاحة

```bash
# التطوير
pnpm dev              # تشغيل الخادم في وضع التطوير

# البناء
pnpm build            # بناء الموقع للإنتاج

# الإنتاج
pnpm start            # تشغيل الموقع المبني

# الاختبارات
pnpm test             # تشغيل اختبارات Vitest

# التحقق
pnpm check            # فحص TypeScript

# التنسيق
pnpm format           # تنسيق الكود مع Prettier

# قاعدة البيانات
pnpm db:push          # تطبيق migrations قاعدة البيانات
```

## 📝 المتغيرات البيئية
